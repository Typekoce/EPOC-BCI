Emotiv EPOC BCI Project

This project enables interfacing with an old Emotiv EPOC EEG brain scanner on Ubuntu Linux. It supports both the modern Cortex API (preferred where compatible) and the legacy reverse-engineered python-emotiv library.

Core Goals

  • Stream raw EEG data in real-time
  • Support mental commands, focus metrics, etc. via Cortex
  • Future integrations: Meta Quest VR, Apple Vision Pro, Unity/Unreal, Android, Wi-Fi/IoT, and more
  • Zero heavy frameworks — pure Python + WebSockets

Current status (Feb 2026): Cortex Linux support is Beta on Ubuntu 22.04+ (64-bit). Old EPOC may require the legacy path if not detected.

Installation Guide

Start by running the automated installer script (legacy path first; Cortex requires separate download).

# Make executable and run
chmod +x install-emotiv-bci.sh
./install-emotiv-bci.sh

This installs: Python 3 deps, git, libusb, bluez, pyusb, pycrypto, numpy, etc., and the python-emotiv library.

Check installed packages and processes at the end of the script output.

Cortex API Path Recommended

1. Download Emotiv Launcher (Ubuntu Beta) from emotiv.com — supports Ubuntu 22.04+ (64-bit).

2. Register a Cortex app → get client_id & client_secret.

3. Edit emotiv_interface.py:

client_id = 'your_client_id'
client_secret = 'your_client_secret'

4. Run with Launcher active:

python3 emotiv_interface.py

Expect live EEG packets printed (timestamps + sensor values).

Legacy python-emotiv Path

Use if old EPOC is not detected by Cortex/Launcher.

After running install script, test:

python3 -m emotiv.epoc

Should show raw sensor packets when headset is on and dongle connected.

Note: Library is archived/unmaintained → fallback only.

Integration Roadmap

Phased plan to extend BCI control to other platforms:

  • Phase 1 (1-2 mo): Research, core command prototypes
  • Phase 2 (2-3 mo): Modular streaming (LSL, WebSockets)
  • Phase 3 (3-6 mo): Platform integrations
    • Meta Quest (Unity plugin)
    • Apple Vision Pro (HID/Unity bridge)
    • Android (Emotiv SDK)
    • Unity 3D & Unreal Engine
    • Nintendo Switch (emulation, limited)
    • Wi-Fi/IoT (Node-RED bridge)
  • Phase 4: Testing, optimization, security
  • Phase 5: Deployment & maintenance (ongoing)

Total estimated: 8-16 months. Start with Unity + Quest for high impact.

Troubleshooting

  • No data? Check saline sensors, fit, dongle LED.
  • Cortex fails? Ensure Launcher running, headset paired, credentials correct.
  • Udev/permissions? Manually copy 99-emotiv.rules and reload udev.
  • Old EPOC not detected? Use legacy path or test in VM with USB passthrough.

See README.md for full details and Cortex docs: emotiv.gitbook.io/cortex-api
